1. PHP: Arrays

1.1. Conceito, Estrutura e Declaração

Arrays em PHP são estruturas de dados que permitem armazenar múltiplos valores em uma única variável. Eles podem ser:

  • Indexados (Numéricos): Os elementos são acessados por um índice numérico (começando do 0).
  • Associativos: Os elementos são acessados por chaves nomeadas (strings).
  • Multidimensionais: Arrays que contêm outros arrays.

Declaração:


<?php
// Array Indexado
$frutas = array("Maçã", "Banana", "Laranja");
// ou (sintaxe curta, mais comum)
$cores = ["Vermelho", "Verde", "Azul"];

// Array Associativo
$pessoa = array(
    "nome" => "Maria",
    "idade" => 30,
    "cidade" => "São Paulo"
);
// ou
$produto = [
    "id" => 101,
    "descricao" => "Notebook",
    "preco" => 2500.00
];

// Array Multidimensional
$alunos = [
    ["nome" => "João", "nota" => 8.5],
    ["nome" => "Ana", "nota" => 9.2],
    ["nome" => "Pedro", "nota" => 7.0]
];
?>

1.2. CRUD em Arrays

As operações básicas (Criar, Ler, Atualizar, Deletar) em arrays PHP são simples:

Criar (Adicionar Elementos)


<?php
$frutas = ["Maçã", "Banana"];
$frutas[] = "Morango"; // Adiciona no final (indexado)
// $frutas agora é ["Maçã", "Banana", "Morango"]

$pessoa = ["nome" => "Carlos"];
$pessoa["idade"] = 25; // Adiciona/atualiza chave (associativo)
// $pessoa agora é ["nome" => "Carlos", "idade" => 25]
?>

Ler (Acessar Elementos)


<?php
$frutas = ["Maçã", "Banana", "Laranja"];
echo $frutas[0]; // Saída: Maçã

$pessoa = ["nome" => "Maria", "idade" => 30];
echo $pessoa["nome"]; // Saída: Maria

// Acessando em array multidimensional
$alunos = [["nome" => "João", "nota" => 8.5]];
echo $alunos[0]["nome"]; // Saída: João
?>

Atualizar (Modificar Elementos)


<?php
$frutas = ["Maçã", "Banana", "Laranja"];
$frutas[1] = "Pera"; // Atualiza o elemento no índice 1
// $frutas agora é ["Maçã", "Pera", "Laranja"]

$pessoa = ["nome" => "Maria", "idade" => 30];
$pessoa["idade"] = 31; // Atualiza o valor da chave 'idade'
// $pessoa agora é ["nome" => "Maria", "idade" => 31]
?>

Deletar (Remover Elementos)


<?php
$frutas = ["Maçã", "Banana", "Laranja"];
unset($frutas[1]); // Remove o elemento no índice 1
// $frutas agora é ["Maçã", "Laranja"] - Note que o índice 1 não existe mais, o 2 continua 2

$pessoa = ["nome" => "Maria", "idade" => 30];
unset($pessoa["idade"]); // Remove a chave 'idade'
// $pessoa agora é ["nome" => "Maria"]
?>

1.3. Ordenação (Funções Prontas)

PHP oferece diversas funções para ordenar arrays:

  • sort($array): Ordena um array indexado em ordem crescente (valores).
  • rsort($array): Ordena um array indexado em ordem decrescente (valores).
  • asort($array): Ordena um array associativo em ordem crescente (valores), mantendo a associação chave/valor.
  • arsort($array): Ordena um array associativo em ordem decrescente (valores), mantendo a associação chave/valor.
  • ksort($array): Ordena um array associativo em ordem crescente (chaves).
  • krsort($array): Ordena um array associativo em ordem decrescente (chaves).

<?php
$numeros = [4, 2, 8, 1];
sort($numeros);
// $numeros agora é [1, 2, 4, 8]

$idades = ["João" => 30, "Maria" => 25, "Carlos" => 35];
asort($idades); // Ordena por valor (idade)
// $idades agora é ["Maria" => 25, "João" => 30, "Carlos" => 35]

ksort($idades); // Ordena por chave (nome)
// $idades agora é ["Carlos" => 35, "João" => 30, "Maria" => 25]
?>

1.4. FOREACH e PRINT_R com Arrays

foreach: É a maneira mais comum e eficiente de iterar sobre arrays.


<?php
$frutas = ["Maçã", "Banana", "Laranja"];
echo "<h4>Frutas:</h4><ul>";
foreach ($frutas as $fruta) {
    echo "<li>" . $fruta . "</li>";
}
echo "</ul>";

$pessoa = ["nome" => "João", "idade" => 30, "cidade" => "Rio"];
echo "<h4>Dados da Pessoa:</h4><ul>";
foreach ($pessoa as $chave => $valor) {
    echo "<li>" . ucfirst($chave) . ": " . $valor . "</li>";
}
echo "</ul>";
?>

print_r(): Uma função útil para imprimir informações legíveis sobre uma variável, especialmente arrays, para depuração.


<?php
$dados = ["item1" => "Valor A", "item2" => "Valor B"];
echo "<pre>"; // Formata a saída para melhor legibilidade
print_r($dados);
echo "</pre>";
/* Saída esperada:
Array
(
    [item1] => Valor A
    [item2] => Valor B
)
*/
?>

1.5. Exemplos

Exemplo completo de manipulação de array:


<?php
// 1. Declaração
$produtos = [
    ["id" => 1, "nome" => "Teclado", "preco" => 150.00],
    ["id" => 2, "nome" => "Mouse", "preco" => 80.00],
    ["id" => 3, "nome" "Monitor", "preco" => 700.00]
];

echo "<h4>Produtos Iniciais:</h4><pre>";
print_r($produtos);
echo "</pre>";

// 2. Adicionar (Create)
$produtos[] = ["id" => 4, "nome" => "Webcam", "preco" => 120.00];
echo "<h4>Após Adicionar Webcam:</h4><pre>";
print_r($produtos);
echo "</pre>";

// 3. Atualizar (Update) - Aumentar preço do Mouse
foreach ($produtos as &$produto) { // Usar & para modificar o array original
    if ($produto["nome"] == "Mouse") {
        $produto["preco"] = 90.00;
        break; // Encontrou e atualizou, pode sair do loop
    }
}
unset($produto); // Desreferenciar a variável para evitar efeitos colaterais
echo "<h4>Após Atualizar Preço do Mouse:</h4><pre>";
print_r($produtos);
echo "</pre>";

// 4. Ler (Read) - Listar produtos com preço > 100
echo "<h4>Produtos com Preço > 100:</h4><ul>";
foreach ($produtos as $produto) {
    if ($produto["preco"] > 100.00) {
        echo "<li>" . $produto["nome"] . " - R$" . number_format($produto["preco"], 2, ',', '.') . "</li>";
    }
}
echo "</ul>";

// 5. Deletar (Delete) - Remover Monitor
foreach ($produtos as $chave => $produto) {
    if ($produto["nome"] == "Monitor") {
        unset($produtos[$chave]);
        break;
    }
}
echo "<h4>Após Remover Monitor:</h4><pre>";
print_r($produtos);
echo "</pre>";

// 6. Ordenar por nome (ksort para associativo, mas aqui é array de arrays, então preciso de usort)
// Para ordenar array de arrays por uma chave interna, usa-se usort
usort($produtos, function($a, $b) {
    return $a['nome'] <=> $b['nome'];
});
echo "<h4>Após Ordenar por Nome:</h4><pre>";
print_r($produtos);
echo "</pre>";
?>

2. HTML: Geral

As tags HTML são a espinha dorsal de qualquer página web. Elas definem a estrutura e o conteúdo. Aqui estão 20 das mais fundamentais e frequentemente usadas:

  1. <!DOCTYPE html>
  2. <html>
  3. <head>
  4. <body>
  5. <title>
  6. <meta>
  7. <link>
  8. <script>
  9. <h1> a <h6>
  10. <p>
  11. <a>
  12. <img>
  13. <div>
  14. <span>
  15. <ul>
  16. <ol>
  17. <li>
  18. <table>
  19. <tr>
  20. <td>

2.2. Ordem da Estrutura da Página HTML

A estrutura básica de uma página HTML segue uma ordem lógica e é essencial para que o navegador interprete e exiba o conteúdo corretamente:


<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título da Minha Página</title>
    <link rel="stylesheet" href="estilos.css">
    <script src="script.js" defer></script>
</head>
<body>
    <!-- Conteúdo visível da página aqui -->
    <h1>Bem-vindo!</h1>
    <p>Este é um parágrafo de exemplo.</p>
</body>
</html>
  • <!DOCTYPE html>: Declaração que define o tipo de documento e a versão do HTML (HTML5). Deve ser a primeira linha.
  • <html lang="pt-BR">: O elemento raiz de toda a página HTML. O atributo lang especifica o idioma do conteúdo.
  • <head>: Contém metadados sobre a página (informações para o navegador e mecanismos de busca), como título, links para CSS, scripts, etc. O conteúdo do <head> não é visível no navegador.
  • <body>: Contém todo o conteúdo visível da página, como textos, imagens, links, formulários, etc.

2.3. Explicação sobre Cada uma das 20 Tags

  1. <!DOCTYPE html>: Define a versão do HTML (HTML5).
  2. <html>: O elemento raiz de uma página HTML.
  3. <head>: Contém metadados e links para recursos externos (CSS, JS).
  4. <body>: Contém o conteúdo visível da página.
  5. <title>: Define o título que aparece na aba do navegador.
  6. <meta>: Usado para metadados (charset, viewport, descrição). Ex: <meta charset="UTF-8">.
  7. <link>: Linka arquivos externos, como folhas de estilo CSS. Ex: <link rel="stylesheet" href="style.css">.
  8. <script>: Incorpora ou linka código JavaScript. Ex: <script src="script.js"></script>.
  9. <h1> a <h6>: Títulos de diferentes níveis (h1 o mais importante, h6 o menos).
  10. <p>: Define um parágrafo de texto.
  11. <a>: Cria um hiperlink para outra página ou recurso. Ex: <a href="pagina.html">Link</a>.
  12. <img>: Incorpora uma imagem. Ex: <img src="imagem.jpg" alt="Descrição">.
  13. <div>: Um contêiner genérico para agrupar elementos, usado para layout com CSS.
  14. <span>: Um contêiner genérico em linha para aplicar estilos a uma parte do texto.
  15. <ul>: Cria uma lista não ordenada (com marcadores).
  16. <ol>: Cria uma lista ordenada (com números ou letras).
  17. <li>: Define um item de lista, usado dentro de <ul> ou <ol>.
  18. <table>: Cria uma tabela.
  19. <tr>: Define uma linha de tabela.
  20. <td>: Define uma célula de dados em uma tabela.
3. HTML: Formulário

Formulários HTML são usados para coletar entrada do usuário. Eles são essenciais para interatividade em sites, permitindo que os usuários enviem dados, façam login, preencham cadastros, etc.

3.1. Os 20 Itens que Compõem o Formulário

Um formulário é composto por diversos elementos de entrada e controle. Aqui estão 20 dos mais comuns:

  1. <form>
  2. <input type="text">
  3. <input type="password">
  4. <input type="email">
  5. <input type="number">
  6. <input type="checkbox">
  7. <input type="radio">
  8. <input type="submit">
  9. <input type="reset">
  10. <input type="hidden">
  11. <input type="file">
  12. <input type="date">
  13. <input type="color">
  14. <textarea>
  15. <select>
  16. <option>
  17. <button>
  18. <label>
  19. <fieldset>
  20. <legend>

3.2. Ordem da Estrutura de Cada Item

A estrutura de um formulário começa com a tag <form>, que define o início e o fim do formulário e seus atributos de envio. Dentro dela, vêm os elementos de entrada.


<form action="processa_dados.php" method="POST">
    <fieldset>
        <legend>Informações Pessoais</legend>
        
        <label for="nome">Nome:</label>
        <input type="text" id="nome" name="nome" placeholder="Seu nome completo" required><br><br>

        <label for="email">Email:</label>
        <input type="email" id="email" name="email" required><br><br>

        <label for="senha">Senha:</label>
        <input type="password" id="senha" name="senha" minlength="6"><br><br>

        <p>Gênero:</p>
        <input type="radio" id="masculino" name="genero" value="M">
        <label for="masculino">Masculino</label><br>
        <input type="radio" id="feminino" name="genero" value="F">
        <label for="feminino">Feminino</label><br><br>

        <label for="mensagem">Sua Mensagem:</label><br>
        <textarea id="mensagem" name="mensagem" rows="5" cols="40"></textarea><br><br>

        <label for="pais">País:</label>
        <select id="pais" name="pais">
            <option value="">Selecione...</option>
            <option value="BR">Brasil</option>
            <option value="PT">Portugal</option>
        </select><br><br>

        <input type="checkbox" id="termos" name="termos" value="aceito" required>
        <label for="termos">Concordo com os termos</label><br><br>

        <button type="submit">Enviar Cadastro</button>
        <button type="reset">Limpar Formulário</button>
    </fieldset>
</form>

3.3. Explicação sobre Cada um dos 20 Componentes

  1. <form>: Define um formulário HTML. Atributos importantes: action (URL de destino) e method (GET ou POST).
  2. <input type="text">: Campo de entrada de texto de linha única.
  3. <input type="password">: Campo de entrada de senha (caracteres ocultos).
  4. <input type="email">: Campo para endereço de e-mail (validação básica de formato).
  5. <input type="number">: Campo para entrada de números.
  6. <input type="checkbox">: Caixa de seleção (permite múltiplas escolhas).
  7. <input type="radio">: Botão de rádio (permite apenas uma escolha em um grupo com o mesmo name).
  8. <input type="submit">: Botão para enviar o formulário.
  9. <input type="reset">: Botão para redefinir os campos do formulário para seus valores iniciais.
  10. <input type="hidden">: Campo invisível para o usuário, usado para enviar dados que não precisam ser exibidos ou modificados.
  11. <input type="file">: Campo para seleção de arquivos para upload.
  12. <input type="date">: Campo para seleção de data.
  13. <input type="color">: Campo para seleção de cor.
  14. <textarea>: Campo de texto de múltiplas linhas. Atributos: rows (altura) e cols (largura).
  15. <select>: Cria uma lista suspensa (dropdown).
  16. <option>: Define uma opção dentro de um <select>. Atributo value é o que será enviado.
  17. <button>: Um botão clicável. Pode ser type="submit", type="reset" ou type="button".
  18. <label>: Define um rótulo para um elemento de formulário. O atributo for deve corresponder ao id do input.
  19. <fieldset>: Agrupa elementos relacionados em um formulário.
  20. <legend>: Fornece uma legenda para o conteúdo de um <fieldset>.

4. PHP: Manipulação de Formulários e Banco de Dados

O PHP é amplamente utilizado para processar dados de formulários HTML e interagir com bancos de dados, formando a base de aplicações web dinâmicas.

4.1. Destino do Formulário é a Página de Recepção dos Dados

O atributo action da tag <form> define para qual URL os dados do formulário serão enviados quando o botão de submit for clicado. Essa URL geralmente aponta para um script PHP que irá processar esses dados.


<form action="processar_cadastro.php" method="POST">
    <!-- Campos do formulário -->
    <input type="text" name="usuario">
    <button type="submit">Enviar</button>
</form>

Neste exemplo, processar_cadastro.php é o script PHP que receberá e manipulará os dados.

4.2. Variável $_POST é um ARRAY

Quando um formulário é enviado usando o método POST, o PHP automaticamente coleta todos os dados dos campos do formulário e os armazena na superglobal $_POST. Esta é uma matriz associativa onde as chaves são os atributos name dos campos do formulário e os valores são os dados inseridos pelo usuário.


<?php
// Conteúdo de processar_cadastro.php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    echo "<h4>Conteúdo de $_POST:</h4><pre>";
    print_r($_POST);
    echo "</pre>";
    /* Saída esperada se o campo 'usuario' foi preenchido com 'Alice':
    Array
    (
        [usuario] => Alice
    )
    */
}
?>

Se o método fosse GET, os dados estariam na superglobal $_GET e seriam visíveis na URL.

4.3. Atribuição às Variáveis dos Valores Vindos do Formulário

É uma boa prática atribuir os valores de $_POST (ou $_GET) a variáveis locais para facilitar o uso e melhorar a legibilidade do código.


<?php
// Conteúdo de processar_cadastro.php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $nome_usuario = $_POST['usuario'];
    $email_usuario = $_POST['email'];
    
    echo "Nome do Usuário: " . $nome_usuario . "<br>";
    echo "Email do Usuário: " . $email_usuario . "<br>";
}
?>

4.4. Tratamento e Manipulação das Variáveis Criadas

Antes de usar os dados do formulário (especialmente em consultas de banco de dados), é crucial tratá-los para segurança e consistência. Isso inclui sanitização (remover caracteres indesejados) e validação (verificar se os dados estão no formato correto).


<?php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    // Sanitização: remove tags HTML e converte caracteres especiais em entidades HTML
    $nome_usuario = htmlspecialchars(trim($_POST['usuario']));
    $email_usuario = filter_var($_POST['email'], FILTER_SANITIZE_EMAIL);
    $idade = (int)$_POST['idade']; // Converte para inteiro

    // Validação básica
    if (empty($nome_usuario)) {
        echo "O nome do usuário é obrigatório.<br>";
    }
    if (!filter_var($email_usuario, FILTER_VALIDATE_EMAIL)) {
        echo "O email informado não é válido.<br>";
    }
    if ($idade < 18) {
        echo "Você deve ter pelo menos 18 anos.<br>";
    }

    // Exemplo de manipulação: converter para maiúsculas
    $nome_usuario_maiusculo = strtoupper($nome_usuario);
    echo "Nome em maiúsculas: " . $nome_usuario_maiusculo . "<br>";
}
?>

4.5. Acesso Autorizado ao Banco

Para interagir com um banco de dados, o PHP precisa de credenciais e uma conexão. A abordagem recomendada é usar PDO (PHP Data Objects) para uma camada de abstração de banco de dados segura e flexível.


<?php
$host = 'localhost';
$db   = 'minha_loja';
$user = 'root';

4.6. CRUD Através do PHP (Comandos SQL)

Com uma conexão PDO estabelecida, você pode executar comandos SQL para realizar operações CRUD. É altamente recomendado usar Prepared Statements para prevenir ataques de Injeção SQL.

Criar (INSERT)


<?php
// Supondo que $pdo já está conectado
$nome_produto = "Smartwatch";
$preco_produto = 350.00;

$sql = "INSERT INTO Produtos (nome, preco) VALUES (:nome, :preco)";
$stmt = $pdo->prepare($sql);
$stmt->execute(['nome' => $nome_produto, 'preco' => $preco_produto]);

echo "Produto '{$nome_produto}' inserido com sucesso!<br>";
?>

Ler (SELECT)


<?php
// Supondo que $pdo já está conectado
$sql = "SELECT id, nome, preco FROM Produtos WHERE preco > :min_preco";
$stmt = $pdo->prepare($sql);
$stmt->execute(['min_preco' => 100.00]);

$produtos = $stmt->fetchAll();

echo "<h4>Produtos com preço > 100:</h4><ul>";
foreach ($produtos as $produto) {
    echo "<li>ID: {$produto['id']}, Nome: {$produto['nome']}, Preço: R$" . number_format($produto['preco'], 2, ',', '.') . "</li>";
}
echo "</ul>";
?>

Atualizar (UPDATE)


<?php
// Supondo que $pdo já está conectado
$novo_preco = 1400.00;
$id_produto = 1;

$sql = "UPDATE Produtos SET preco = :preco WHERE id = :id";
$stmt = $pdo->prepare($sql);
$stmt->execute(['preco' => $novo_preco, 'id' => $id_produto]);

echo "Produto com ID {$id_produto} atualizado com sucesso!<br>";
?>

Deletar (DELETE)


<?php
// Supondo que $pdo já está conectado
$id_produto_deletar = 2;

$sql = "DELETE FROM Produtos WHERE id = :id";
$stmt = $pdo->prepare($sql);

?>

Exemplo Completo: Formulário de Cadastro de Usuário e Inserção no Banco de Dados

Arquivo: cadastro.html


<!DOCTYPE html>
<html lang="pt-BR">

        <button type="submit">Cadastrar</button>

Arquivo: processar_cadastro.php


<?php
// 1. Configuração da Conexão com o Banco de Dados
$host = 'localhost';
$db   = 'minha_aplicacao'; // Certifique-se de que este banco de dados existe
$user = 'root';
$pass = ''; // Sua senha do MySQL
$charset = 'utf8mb4';

$dsn = "mysql:host=$host;dbname=$db;charset=$charset";
$options = [
    PDO::ATTR_ERRMODE            => PDO::ERRMODE_EXCEPTION,
    PDO::ATTR_DEFAULT_FETCH_MODE => PDO::FETCH_ASSOC,
    PDO::ATTR_EMULATE_PREPARES   => false,
];

try {
    $pdo = new PDO($dsn, $user, $pass, $options);
} catch (\PDOException $e) {
    die("Erro de Conexão com o Banco de Dados: " . $e->getMessage());
}

// 2. Processamento do Formulário
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    // 3. Atribuição e Tratamento das Variáveis
    $nome = htmlspecialchars(trim($_POST['nome']));
    $email = filter_var($_POST['email'], FILTER_SANITIZE_EMAIL);
    $senha = password_hash($_POST['senha'], PASSWORD_DEFAULT); // Hash da senha para segurança!

    // Validação
    if (empty($nome) || empty($email) || empty($_POST['senha'])) {
        die("Por favor, preencha todos os campos obrigatórios.");
    }
    if (!filter_var($email, FILTER_VALIDATE_EMAIL)) {
        die("Email inválido.");
    }

    // 4. Inserção no Banco de Dados (CREATE)
    try {
        $sql = "INSERT INTO usuarios (nome, email, senha) VALUES (:nome, :email, :senha)";
        $stmt = $pdo->prepare($sql);
        $stmt->execute([
            'nome' => $nome,
            'email' => $email,
            'senha' => $senha
        ]);
        echo "<h3>Usuário '{$nome}' cadastrado com sucesso!</h3>";

        // Opcional: Exibir todos os usuários após o cadastro (READ)
        echo "<h4>Usuários Cadastrados:</h4>";
        $stmt_select = $pdo->query("SELECT id, nome, email FROM usuarios");
        $usuarios = $stmt_select->fetchAll();

        if (count($usuarios) > 0) {
            echo "<table border='1'>";
            echo "<thead><tr><th>ID</th><th>Nome</th><th>Email</th></tr></thead>";
            echo "<tbody>";
            foreach ($usuarios as $usuario) {
                echo "<tr>";
                echo "<td>" . htmlspecialchars($usuario['id']) . "</td>";
                echo "<td>" . htmlspecialchars($usuario['nome']) . "</td>";
                echo "<td>" . htmlspecialchars($usuario['email']) . "</td>";
                echo "</tr>";
            }
            echo "</tbody>";
            echo "</table>";
        } else {
            echo "<p>Nenhum usuário cadastrado ainda.</p>";
        }

    } catch (\PDOException $e) {
        // Erro de duplicação de email, por exemplo
        if ($e->getCode() == 23000) { // Código SQLSTATE para violação de integridade
            die("Erro: O email '{$email}' já está cadastrado.");
        }
        die("Erro ao cadastrar usuário: " . $e->getMessage());
    }
} else {
    echo "<p>Acesso inválido. Por favor, envie o formulário.</p>";
}
?>

Nota: Para este exemplo funcionar, você precisaria de um banco de dados MySQL (ou similar) chamado minha_aplicacao e uma tabela usuarios com colunas id INT PRIMARY KEY AUTO_INCREMENT, nome VARCHAR(100), email VARCHAR(100) UNIQUE e senha VARCHAR(255).
