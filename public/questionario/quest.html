1. PHP: Arrays

1.1. Conceito, Estrutura e Declaração

Questão 1
Você está desenvolvendo um sistema de gerenciamento de estoque e precisa armazenar uma lista de produtos, onde cada produto tem um nome e um preço. Qual a melhor forma de declarar um array em PHP para representar essa estrutura, permitindo fácil acesso ao nome e preço de cada produto?

  • A) `$produtos = ["Teclado", 150.00, "Mouse", 80.00];`
  • B) `$produtos = [["nome" => "Teclado", "preco" => 150.00], ["nome" => "Mouse", "preco" => 80.00]];`
  • C) `$produtos = "Teclado,150.00;Mouse,80.00";`
  • D) `$produtos = new Array("Teclado", "Mouse");`

Questão 2
Em um projeto PHP, você precisa armazenar os dias da semana em ordem. Qual tipo de array PHP seria o mais natural e eficiente para essa finalidade, considerando que você acessaria os dias por sua posição numérica (ex: 0 para Domingo, 1 para Segunda)?

  • B) Array Multidimensional
  • C) Array Indexado

  • C) `["elemento1", "elemento2"]`
  • D) `list("elemento1", "elemento2")`

Questão 4

1.4. FOREACH e PRINT_R com Arrays

Questão 16

  • A) `for ($i = 0; $i < count($itens); $i++)`
  • B) `while ($item = current($itens))`
  • C) `foreach ($itens as $item)`
  • D) `do { ... } while (true)`

Questão 17

  • A) `echo $dadosUsuario;`
  • B) `var_dump($dadosUsuario);`
  • C) `print_r($dadosUsuario);`
  • D) `json_encode($dadosUsuario);`

Questão 18

  • C) `foreach ($produto as $key)`
  • D) `foreach ($produto as $index, $item)`

  • B) `foreach` permite modificar os elementos do array diretamente sem usar referência.
  • C) `foreach` simplifica a iteração, abstraindo o acesso por índice ou chave.
  • D) `foreach` é a única forma de iterar sobre arrays multidimensionais.

Questão 20
Você tem um array `$cores = ["red", "green", "blue"];` e quer imprimir cada cor em uma nova linha. Qual trecho de código PHP faria isso corretamente?

  • A) `echo implode("\n", $cores);`

  • C) `foreach ($cores as $cor) { echo $cor . "\n"; }`
  • D) Todas as alternativas A, B e C estão corretas.
